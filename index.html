البرمجة

هي عملية إنشاء أوامر وتعليمات تُنفذ بواسطة الحواسيب.

البرمجة هي عملية كتابة تعليمات وأوامر بلغة يفهمها الحاسوب، من أجل تنفيذ مهام محددة أو إنشاء برامج وتطبيقات. تُستخدم البرمجة في: تطوير مواقع الويب والتطبيقات تصميم الألعاب التحكم في الآلات والروبوتات تحليل البيانات بناء أنظمة الذكاء الاصطناعي 👨‍💻 من أشهر لغات البرمجة: Python JavaScript C++ Java PHP المبرمج يكتب الكود، ثم يختبره ويقوم بتحسينه ليعمل بكفاءة ويحل مشكلة معينة.

HTML

لغة ترميزية تُستخدم لبناء صفحات الويب.

HTML (اختصار لـ HyperText Markup Language) هي لغة ترميز تُستخدم لإنشاء الهيكل الأساسي لصفحات الويب. 🔸 تسمح HTML بتحديد عناصر الصفحة مثل: العناوين الفقرات الصور الروابط الجداول القوائم 🚫 ملاحظة: HTML ليست لغة برمجة، بل لغة وصفية (Markup Language)، أي تُستخدم فقط لوصف وتنظيم المحتوى، ولا تنفّذ أوامر برمجية.

paython

مفهوم بايثون

بايثون هي لغة برمجة عالية المستوى، سهلة التعلم، وقوية في الأداء. تُستخدم في تطوير المواقع، الذكاء الاصطناعي، تحليل البيانات، برمجة الألعاب، الأتمتة، وغيرها من المجالات. تم إنشاؤها في أواخر الثمانينات من قبل Guido van Rossum، وهي اليوم من أكثر اللغات استخدامًا حول العالم. مميزات لغة بايثون: ✅ سهلة القراءة والكتابة (تشبه اللغة الإنجليزية). ✅ مجانية ومفتوحة المصدر. ✅ تدعم البرمجة الكائنية (OOP) والبرمجة الإجرائية. ✅ توفر آلاف المكتبات الجاهزة (مثل NumPy، Pandas، Flask، Django...). ✅ متعددة الاستخدامات (تطبيقات ويب، ذكاء اصطناعي، روبوتات، تحليل بيانات...). استخدامات بايثون: المجال الاستخدام تطوير الويب باستخدام Django أو Flask تحليل البيانات باستخدام Pandas و Matplotlib الذكاء الاصطناعي باستخدام TensorFlow و PyTorch الروبوتات التحكم بالحساسات والمحرّكات البرمجة التعليمية لغة مثالية للمبتدئين بسبب بساطتها أتمتة المهام برمجة سكريبتات لأداء مهام متكررة تلقائيًا مثال بسيط على كود بايثون: python Copier Modifier # طباعة رسالة ترحيبية name = input("ما اسمك؟ ") print("مرحبًا، " + name + "! سعيد بلقائك.") لماذا أتعلم بايثون؟ مطلوبة في سوق العمل بشدة. تدخل في كل المجالات التقنية تقريبًا. تساعدك على تحويل أفكارك إلى مشاريع حقيقية.

C++

مفهوم c++

C++ هي لغة برمجة قوية ومرنة، تُستخدم في تطوير الأنظمة، الألعاب، التطبيقات عالية الأداء، والبرمجيات التي تحتاج إلى تحكم مباشر في موارد الجهاز مثل الذاكرة والمعالج. تم تطويرها في أوائل الثمانينات على يد Bjarne Stroustrup، وتُعتبر امتدادًا للغة C بإضافة مفاهيم البرمجة الكائنية (Object-Oriented Programming). مميزات لغة C++: ⚡ أداء عالي جدًا (قريبة من لغة الآلة). ⚙️ تحكم مباشر بالذاكرة عبر المؤشرات. 🧱 تدعم البرمجة الكائنية (OOP): الكائنات، الوراثة، التغليف... 🔄 تدعم البرمجة الإجرائية (مثل C). 🔌 تُستخدم لتطوير الأنظمة (مثل أنظمة التشغيل وبرامج التشغيل). 🎮 مستخدمة بكثرة في برمجة الألعاب ومحركات الألعاب (مثل Unreal Engine). استخدامات C++: المجال الاستخدامات أنظمة التشغيل تطوير أجزاء من Windows، Linux، وغيرها محركات الألعاب Unreal Engine مكتوبة بـ C++ البرمجيات المكتبية مثل متصفحات، برامج تحرير... الروبوتات والأنظمة المدمجة تحكم مباشر بالهاردوير الخوارزميات والأداء العالي مثل المحاكيات والتطبيقات العلمية مثال بسيط على كود C++: cpp Copier Modifier #include using namespace std; int main() { string name; cout << "ما اسمك؟ "; cin >> name; cout << "مرحبًا، " << name << "!" << endl; return 0; } الفرق بين C و C++: الخاصية C C++ النمط إجرائي فقط إجرائي + كائني الكائنات غير مدعومة مدعومة التعقيد أبسط أكثر تعقيدًا لكن أقوى الأداء عالي جدًا عالي جدًا + ميزات إضافية.

javascript

مفهوم javascript

JavaScript هي لغة برمجة عالية المستوى تُستخدم أساسًا لتطوير واجهات المواقع الإلكترونية التفاعلية. وهي واحدة من الركائز الثلاث لتطوير الويب إلى جانب HTML وCSS. تعمل JavaScript في المتصفح مباشرة، مما يجعلها مثالية لإنشاء صفحات ويب ديناميكية وحيوية. مميزات JavaScript: ⚡ تشغيل مباشر في المتصفح دون الحاجة لتثبيت. 🖱️ تفاعل حي مع المستخدم (الأزرار، النماذج، الحركات). 🔄 ديناميكية وسهلة التعلّم. 🔌 مدعومة في جميع المتصفحات الحديثة. 🧠 تُستخدم مع مكتبات وأُطر مثل React، Vue، Angular لتطوير تطبيقات ضخمة. 🌍 يمكن استخدامها أيضًا في السيرفرات عبر Node.js. لماذا أتعلم JavaScript؟ 🔥 أساس لأي مطوّر ويب. 📱 تفتح لك أبوابًا لبناء تطبيقات الهاتف والمواقع التفاعلية. 💼 مطلوبة بشدة في سوق العمل. 🌐 تُستخدم في الواجهة الأمامية والخلفية (Frontend & Backend).

java

مفهوم java

Java هي لغة برمجة كائنية التوجّه (Object-Oriented)، عالية المستوى، تم تطويرها من قبل شركة Sun Microsystems في التسعينات، وتدعمها الآن شركة Oracle. تتميز Java بشعارها الشهير: "Write Once, Run Anywhere" (اكتبها مرة، وشغّلها في أي مكان) وهذا يعني أن البرامج المكتوبة بلغة Java تعمل على أي نظام تشغيل دون تعديل، طالما أن Java Virtual Machine (JVM) متوفرة عليه. مميزات لغة Java: ☕ محمولة (Portable): تعمل على أي نظام. 🔐 آمنة (Secure): تحتوي على نظام أمان قوي. ♻️ كائنية التوجّه (OOP): كل شيء يُعبّر عنه بالكائنات. ⚙️ قوية الأداء: تستخدم في تطبيقات ضخمة ومعقدة. 📱 تُستخدم لتطوير تطبيقات الأندرويد. 🌐 تُستخدم في تطبيقات الويب، السيرفرات، المؤسسات البنكية.

كرة السلة

لعبة جماعية تُمارس بين فريقين يحاولان إدخال الكرة في سلة الخصم.

كرة السلة هي رياضة جماعية تُلعب بين فريقين، يتكوّن كل فريق من خمسة لاعبين. 🎯 الهدف من اللعبة هو إدخال الكرة في سلة الخصم التي تكون مُرتفعة عن الأرض، ويُسجّل الفريق نقاطًا حسب عدد الرميات الناجحة. 🔹 أهم قواعد اللعبة: كل فريق يهاجم ويدافع بالتناوب يتم استخدام اليد فقط في تمرير أو تصويب الكرة يُمنع الاحتكاك الجسدي القوي (الفاولات) ⏱ مدة المباراة تتكوّن عادة من أربع أشواط، كل شوط يستمر لمدة 10 أو 12 دقيقة (حسب نظام البطولة). 🏆 من أبرز البطولات: الدوري الأمريكي للمحترفين (NBA) الألعاب الأولمبية بطولات العالم والقارات

كرة الطائرة

لعبة جماعية تُمارس بين فريقينم.

كرة الطائرة هي رياضة جماعية تُلعب بين فريقين، يتكوّن كل فريق من 6 لاعبين. 🎯 الهدف من اللعبة هو تمرير الكرة فوق الشبكة ومحاولة جعلها تلمس أرضية الفريق الخصم، بينما يحاول الفريق الآخر صدّها ومنعها من السقوط. 🔹 قواعد أساسية: كل فريق يُسمح له بثلاث لمسات فقط قبل إرسال الكرة للطرف الآخر يُمنع مسك الكرة أو ضربها أكثر من مرة متتالية من نفس اللاعب يتم تسجيل نقطة عندما تفشل الجهة المقابلة في صدّ الكرة بشكل قانوني ⏱ المباراة تُلعب عادة من 3 أو 5 أشواط، والفريق الذي يفوز بثلاثة أشواط أولًا يُعتبر الفائز. 🌍 كرة الطائرة رياضة أولمبية شهيرة وتُمارس في جميع أنحاء العالم، سواء في الصالات أو على الشاطئ.

كرة اليد

مفهوم كرة اليد

كرة اليد هي رياضة جماعية تُلعب بين فريقين، ويتكون كل فريق من 7 لاعبين (6 لاعبين ميدانيين + حارس مرمى). 🎯 الهدف من اللعبة هو إدخال الكرة في مرمى الفريق الخصم عن طريق رميها باليد، والفريق الذي يُسجّل أكبر عدد من الأهداف هو الفائز. 🔹 قواعد أساسية: يُسمح لكل لاعب بلمس الكرة بيديه فقط لا يُسمح بالجري أكثر من ثلاث خطوات دون تنطيط الكرة يُمنع مسك الكرة أكثر من 3 ثوانٍ دون تمرير أو تسديد التلامس الجسدي محدود ومراقب من الحكام ⏱ تتكون المباراة عادة من شوطين، كل شوط مدته 30 دقيقة، مع استراحة بين الشوطين. ⚽ كرة اليد رياضة سريعة ومليئة بالحماس والتكتيك، وتُمارس في الصالات المغلقة.

المتصفح

برنامج يُستخدم لتصفح مواقع الإنترنت مثل Chrome وFirefox.

المتصفح هو برنامج يُستخدم لفتح مواقع الإنترنت وتصفح محتواها مثل النصوص، الصور، الفيديوهات، والتطبيقات. من أشهر المتصفحات: Google Chrome Mozilla Firefox Microsoft Edge Safari المتصفح يقوم بترجمة لغات الويب مثل HTML وCSS وJavaScript إلى صفحات يمكننا مشاهدتها والتفاعل معها. يحتوي المتصفح على العديد من الخصائص مثل: فتح عدة تبويبات في وقت واحد حفظ كلمات المرور التصفح الخفي دعم الإضافات (Extensions) لتحسين الأداء

الانترنت

مفهوم الانترنت ومكوناته

الإنترنت هو شبكة عالمية ضخمة تربط بين ملايين الأجهزة حول العالم، وتمكّن المستخدمين من تبادل المعلومات والبيانات والتواصل بسهولة وسرعة. يُعد الإنترنت من أهم الابتكارات في العصر الحديث، حيث غيّر جذريًا طريقة تواصل البشر وتبادلهم للمعرفة والخدمات. الخوادم (Servers): أجهزة تقوم بتخزين المعلومات والملفات والمواقع. الأجهزة العميلة (Clients): كالحواسيب والهواتف التي يتصفح بها المستخدم الإنترنت. شبكات الاتصال: مثل الألياف البصرية، الواي فاي، الأقمار الصناعية. بروتوكولات الإنترنت: مثل HTTP، وTCP/IP التي تنظّم عمليات التبادل بين الأجهزة. خدمات يقدمها الإنترنت: تصفح المواقع الإلكترونية. التواصل عبر البريد الإلكتروني ووسائل التواصل الاجتماعي. مشاهدة الفيديوهات والاستماع للموسيقى. التعليم عن بُعد والدورات الإلكترونية. التسوق الإلكتروني. الوصول إلى قواعد البيانات والمعلومات. فوائد الإنترنت: الوصول السريع إلى المعلومات. تسهيل التواصل بين الأفراد حول العالم. دعم التعليم والعمل عن بعد. تعزيز التجارة الإلكترونية. أضرار الإنترنت (عند الاستخدام السيئ): الإدمان الرقمي. انتشار المعلومات الكاذبة. التعرّض للاختراقات الأمنية. العزلة الاجتماعية.

التكنلوجيا

مفهوم التكنلوجيا

التكنولوجيا هي تطبيق المعرفة العلمية لهدف عملي، أي استخدام الأدوات والابتكارات والأساليب لحل المشكلات وتسهيل الحياة. وهي تتطوّر باستمرار لتلبية احتياجات الإنسان في مختلف المجالات، مثل الصحة، التعليم، الصناعة، الاتصالات، والترفيه. أنواع التكنولوجيا: تكنولوجيا المعلومات (IT): مثل الحواسيب والبرمجيات والإنترنت. التكنولوجيا الطبية: مثل أجهزة الأشعة، والروبوتات الجراحية، والتطبيقات الصحية. التكنولوجيا الصناعية: مثل الآلات، وخطوط الإنتاج الأوتوماتيكية. تكنولوجيا الاتصالات: مثل الهواتف، وشبكات الجيل الخامس، وتطبيقات التواصل. التكنولوجيا التعليمية: مثل اللوحات الذكية، والدروس عبر الإنترنت. أهمية التكنولوجيا: تسريع العمل والإنتاج. تسهيل الحياة اليومية والتواصل. تحسين الرعاية الصحية والتعليم. إتاحة الوصول إلى المعرفة والمعلومات. فوائد التكنولوجيا: توفير الوقت والجهد. توسيع فرص التعلم والعمل. تحسين جودة الحياة. دعم الابتكار والتطوير. سلبيات التكنولوجيا (عند الإفراط أو الاستخدام السيئ): الإدمان على الأجهزة. تقليل التفاعل الاجتماعي. تهديد الخصوصية. فقدان بعض الوظائف بسبب الأتمتة.

التجارة الرقمية

مفهوم التجارة الرقمية

التجارة الرقمية (أو التجارة الإلكترونية) هي عملية شراء وبيع السلع أو الخدمات عبر الإنترنت. تعتمد على استخدام التكنولوجيا الرقمية مثل المواقع الإلكترونية، وتطبيقات الهواتف، وأنظمة الدفع الإلكترونية لتسهيل عمليات التجارة دون الحاجة إلى وجود مادي. أشكال التجارة الرقمية: التجارة بين الشركات (B2B): مثل بيع المواد الخام أو البرمجيات بين شركتين. التجارة بين الشركة والمستهلك (B2C): مثل شراء الملابس من متجر إلكتروني. التجارة بين مستهلكين (C2C): مثل بيع المنتجات المستعملة عبر منصات مثل OLX أو eBay. التجارة بين الحكومة والأفراد أو الشركات (G2C / G2B): مثل دفع الفواتير الحكومية عبر الإنترنت. مكونات التجارة الرقمية: المتجر الإلكتروني أو المنصة. منتجات أو خدمات رقمية أو مادية. وسائل الدفع الإلكتروني (بطاقات بنكية، PayPal، الدفع عند الاستلام...). خدمات التوصيل والشحن. التسويق الرقمي لجذب الزبائن. فوائد التجارة الرقمية: الوصول إلى عدد أكبر من الزبائن حول العالم. العمل طوال أيام الأسبوع على مدار الساعة. تقليل التكاليف مقارنة بالتجارة التقليدية. تحليل سلوك المستهلك لتحسين الخدمة. التحديات: المنافسة الشديدة. مشاكل الشحن والتوصيل. حماية البيانات والخصوصية. الثقة بين البائع والمشتري.

المجرة

هي مجموعة ضخمة من النجوم والكواكب والغبار الكوني تدور معًا في الفضاء.

المجرة هي مجموعة ضخمة جدًا من النجوم والكواكب والغازات والغبار الكوني، تدور معًا في الفضاء بفعل الجاذبية. 🔹 يوجد مليارات المجرات في الكون، وتأتي بأشكال مختلفة مثل: المجرات الحلزونية (مثل درب التبانة) المجرات البيضاوية المجرات غير المنتظمة 🌌 مجرة درب التبانة هي المجرة التي نعيش فيها، وتضم أكثر من 100 مليار نجم. ☄️ تحتوي كل مجرة على عدد هائل من الأنظمة الشمسية، ومن ضمنها نظامنا الشمسي الذي يدور فيه كوكب الأرض.

القمر الاستطناعي

القمر الصناعي هو جسم يُطلق إلى الفضاء ويدور حول الأرض أو أي جسم سماوي آخر.

🔸 يستخدم لأغراض متعددة مثل: الاتصالات (كالإنترنت والهاتف) الرصد الجوي والتنبؤ بالطقس التصوير الفضائي والمراقبة الأبحاث العلمية والاستكشاف 🔹 القمر الصناعي قد يكون صغيرًا أو كبيرًا، وهو يعمل باستخدام ألواح شمسية للطاقة وأجهزة إرسال واستقبال. 🚀 مثال: "القمر الصناعي نايل سات" يُستخدم لنقل البث التلفزيوني في الشرق الأوسط.

🔸الكواكب هي أجسام كروية ضخمة تدور حول نجم (مثل الشمس) وتُعتبر جزءًا من النظام الشمسي أو أنظمة نجمية أخرى. 🔹 الكوكب يتميز بأنه: له جاذبية كافية ليأخذ شكلاً كرويًا يدور حول نجم يُزيل الأجسام الصغيرة من مداره لا يُنتج ضوءًا بنفسه بل يعكس ضوء النجم 🌞 في نظامنا الشمسي، هناك ثمانية كواكب رئيسية: عطارد الزهرة الأرض المريخ المشتري زحل أورانوس نبتون 🌍 كوكب الأرض هو الكوكب الوحيد المعروف حاليًا بوجود الحياة عليه.

المثلت

مفهوم المثلت و خصائصه

➗ مفهوم الثلث وخصائصه: الثلث هو جزء من ثلاثة أجزاء متساوية. أي إذا قُسِّم شيء إلى ثلاثة أجزاء متساوية، فإن الثلث هو أحد هذه الأجزاء. 🔹 مثال: إذا قسّمت تفاحة إلى 3 أجزاء متساوية، فإن كل جزء هو "ثلث التفاحة". الكسر الذي يُعبّر عن الثلث هو: ⅓ أو 1/3 ⭐ خصائص الثلث: الثلث + الثلث + الثلث = 1 (الكل) أي: 1 3 + 1 3 + 1 3 = 1 3 1 ​ + 3 1 ​ + 3 1 ​ =1 الثلث أصغر من النصف: لأن 1 3 < 1 2 3 1 ​ < 2 1 ​ يمكن تحويله إلى عدد عشري: 1 3 = 0.333... 3 1 ​ =0.333... (عدد دوري لا ينتهي) يُستخدم كثيرًا في الرياضيات والقياس والطبخ والفن، حيث يساعد في تقسيم الأشياء بالتساوي.

المربع

مفهوم المربع و خصائصه

المربع هو شكل هندسي ثنائي الأبعاد، يتكوّن من أربعة أضلاع متساوية وأربع زوايا قائمة (كل زاوية = 90 درجة). ⭐ خصائص المربع: جميع الأضلاع متساوية الطول إذا كان أحد الأضلاع طوله 5 سم، فكل الأضلاع الأخرى أيضًا 5 سم. جميع الزوايا قائمة أي أن كل زاوية في المربع تساوي 90 درجة. له قطران متساويان القطران يتقاطعان في المنتصف ويشكلان زاوية قائمة. المربع هو حالة خاصة من المستطيل والمعين لأنه يجمع خصائص الشكلين معًا. محيط المربع = 4 × طول الضلع مثال: إذا كان طول الضلع = 6 سم، فالمحيط = 24 سم مساحة المربع = طول الضلع × نفسه مثال: إذا كان طول الضلع = 5 سم، فالمساحة = 25 سم²

مفهومpythagore

فيثاغورس (بالفرنسية: Pythagore – وبالعربية يُنطق أحيانًا "بتا غورس") هو فيلسوف وعالم رياضيات يوناني قديم، وُلد حوالي سنة 570 قبل الميلاد. يُعد من أهم الشخصيات في تاريخ الرياضيات والفلسفة. 📚 أشهر إسهاماته: 🔺 نظرية فيثاغورس: واحدة من أشهر النظريات في الرياضيات، وتنص على: في المثلث القائم الزاوية، مربع طول الوتر = مجموع مربعي طولي الضلعين القائمين. الصيغة الرياضية: 𝑐 2 = 𝑎 2 + 𝑏 2 c 2 =a 2 +b 2 حيث: 𝑐 c: طول الوتر (الضلع المقابل للزاوية القائمة) 𝑎 a و 𝑏 b: طولا الضلعين الآخرين مثال: إذا كان أحد الضلعين = 3، والآخر = 4، فإن: 𝑐 2 = 3 2 + 4 2 = 9 + 16 = 25 ⇒ 𝑐 = 5 c 2 =3 2 +4 2 =9+16=25⇒c=5 ✨ إسهامات أخرى: ربط بين الرياضيات والموسيقى (درس العلاقة بين الطول الموجي والنغمات) آمن بأن الأرقام لها قوى وتأثيرات روحية أسس مدرسة فلسفية تعتمد على الرياضيات كنظام لفهم الكون

الرياضيات

مفهوم الرياضيات

الرياضيات هي علم يدرس الأعداد، الأشكال، الكميات، والعلاقات بينها باستخدام الرموز والقوانين المنطقية. 🔹 تُستخدم الرياضيات في: الحساب اليومي (مثل البيع والشراء) العلوم والهندسة التكنولوجيا والبرمجة تحليل البيانات والإحصاء فهم الظواهر الطبيعية والكونية ⭐ فروع الرياضيات الأساسية: الحساب: جمع، طرح، ضرب، قسمة الجبر: استخدام الرموز للتعبير عن المعادلات والعلاقات الهندسة: دراسة الأشكال والمساحات والحجوم الإحصاء: جمع البيانات وتحليلها التحليل الرياضي: دراسة التغيرات (مثل التفاضل والتكامل) نظرية الأعداد: دراسة خصائص الأعداد (مثل الأولية) 🧠 لماذا الرياضيات مهمة؟ تُنمي التفكير المنطقي تُساعد على حل المشكلات تُستخدم في جميع مجالات الحياة اليومية هي أساس العلوم الحديثة والتكنولوجيا

الرياضيات

مفهوم الرياضيات

الرياضيات هي علم دراسة الكميات، الأعداد، الأشكال، الأنماط، والعلاقات باستخدام الرموز والمنطق. تُعد الرياضيات لغة العلم الأساسية، لأنها تُستخدم في التعبير عن الأفكار بدقة، وتُعد أداة ضرورية في مجالات متعددة مثل الفيزياء، والهندسة، والاقتصاد، وعلوم الحاسوب. فروع الرياضيات الرئيسية: الحساب (Arithmetic): يهتم بالأعداد والعمليات الأساسية (الجمع، الطرح، الضرب، القسمة). الجبر (Algebra): يدرس العلاقات بين الرموز والمتغيرات. الهندسة (Geometry): تركز على الأشكال والمساحات والحجوم. الإحصاء والاحتمالات (Statistics & Probability): دراسة البيانات وتحليلها والتنبؤ بالنتائج. التفاضل والتكامل (Calculus): يهتم بدراسة التغيرات والمعدلات. أهمية الرياضيات: تطوير التفكير المنطقي والتحليلي. أساس للتقدم في التكنولوجيا والهندسة والعلوم. تُستخدم في الحياة اليومية (المال، الوقت، القياس...). أداة لفهم الظواهر الطبيعية والاقتصادية. تطبيقات الرياضيات في الحياة: حساب الميزانية والمصاريف. تصميم المباني والهياكل. تحليل البيانات في الأعمال. تطوير البرمجيات والخوارزميات. فهم الإحصاءات والرسوم البيانية.
